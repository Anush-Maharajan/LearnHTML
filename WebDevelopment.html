HTML Basis

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure webpages.

Essential Tags

"html", "head", "title", "body", "p", "a", "img", "ul", "ol", etc.

CSS Basics

What is CSS?

CSS (Cascading Style Sheets) is a language used to describe the presentation and formatting of HTML documents.

Responsive Design

Responsive design is a web development approach that ensures a website's layout and content adapt to various screen sizes and devices.

JavaScript Basics

What is JAVASCRIPT?

JavaScript (JS) is a programming language commonly used to create interactive effects within web browsers.

DOM Manupilation

DOM manipulation involves using JavaScript to dynamically change the content, structure, and style of a webpage's Document Object Model (DOM).

Frameworks
